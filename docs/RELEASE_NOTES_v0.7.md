# ArIEC103 v0.7 - Superseded Semantic Experiment

v0.7 introduced an early semantic-label experiment. It is superseded by v0.8.

Do not treat v0.7 as the product direction for signal naming.

The correct direction from v0.8 onward is:

- no built-in relay/vendor signal profile
- signal names come only from user/project mapping profile
- raw FUN/INF remains the source of truth
- Value Viewer and Event Log use mapped names only when the user provides mapping
