# ArIEC103 v1.2.21 — Fluid Segmented Navigation + Bright Modern Buttons

## UI motion
- Reworked segmented tab navigation to remove the grey selected state.
- Added a soft sliding-pill feel for the selected tab through animated pill scale/opacity.
- Added tactile tab interactions: hover grows subtly, press compresses subtly.
- Kept the navigation premium and calm; no heavy glow or noisy motion.

## Button palette
- Replaced dark-blue button treatment with a brighter modern blue/cyan accent.
- Button text now remains dark/black for readability.
- Primary rail buttons, primary buttons, and selected segmented tabs now share the brighter AR software accent direction.

## Guardrail
- Buttons and segmented navigation should stay bright, readable, and tactile.
- Avoid dark-blue backgrounds with dark captions.
- Avoid grey selected states on the segmented navbar.
